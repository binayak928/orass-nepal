Project Description

Garuda

Garuda is a 83 inch tall solid rocket that is capable of carrying 4kg Surveillance payload and deploy it at an altitude of 10,000 ft AGL . It has active flight control system for achieving precise altitude.

The Phoenix

The Phoenix is a 60 inch solid sounding rocket designed by ORASS. This is our first rocket project with student researched and developed motors. It will be able to carry 1.5 kg payload to 1.4 km apogee. It has a single parachute deployment mechanism that will help the rocket for safe touch down and reusability. This rocket was designed to compete in Military Science and Technology Competition-II. Due to Covid Pandemic it was not completed on time.

SWIFT SAT

Swift Sat is a can sized satellite model that has a similar working process like that of a real Satellite. This Can-Sat is capable of reading basic atmospheric data like temperature, humidity, air pressure, altitude from the sea level, light Intensity, it’s latitude, longitude and orientation and transmitting them to the ground station. This Project bagged 2nd position in NCIT Expo on Technology under best hardware project.
